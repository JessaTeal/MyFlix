Documentation for My Movie API

My Movie API is a program to made with Javascript to prove competency with Javascript, Modules, and creating/implementing API's.

Return a list of all movies to the user.

Endpoint /movies
HTTP Method GET
Query Parameters NONE
Request Body Format None
Response Data Format A JSON object holding data about all movies.

{
"title": "Sweet Home Alabama",
"imageUrl": "https://flxt.tmsimg.com/assets/p30076_p_v12_aq.jpg",
"genre": ["Comedy", "Romance"]
"director": "Andy Tennant",
"tagline": "Sometimes what you're looking for is right where you left it.",
"description": "A young woman who has reivented herself as a New York City socialite must return home to Alabama to obtain a divorce from her husband after seven years of separation."
}

Return data about a single movie.

Endpoint /movies/:title
HTTP Method GET
Query Parameters :title refers to the title of the specific movie the user is wanting information on.
Request Body Format None
Response Data Format A JSON object holding data about a specific movie. Including title, genre, tagline, and description.

{
"title": "Pitch Perfect",
"imageUrl": "https://m.media-amazon.com/images/I/91bAKJ3lMVL._SY679_.jpg",
"genre": ["Comedy", "Music", "Romance"],
"director": "Jason Moore",
"tagline": "Get Pitch Slapped",
"description": "Becca, a freshman at Barden University, is cajoled into joining the Bellas, her school's all-girls singing group. Injecting some much needed energy into their repertoire, The Bellas take on their male rivals in a campus competition."
},

Return data about a movies genre.

Endpoint /movies/:title/genre
HTTP Method GET
Query Parameters :title refers to the title of the specific movie the user wants genre information for.
Request Body Format None
Response Data Format A JSON object that includes the genre of a specific movie.

"The genre of Pride and Prejudice is: Romance"

Return data about a director of a specific movie.

Endpoint /movies/:title/director
HTTP Method GET
Query Parameters :title refers to the title of the specific movie the user wants director information for.
Request Body Format None
Response Data Format A JSON object that includes the birth year, death year, and description of the director.

"Name: John Smith
Birth Year: 1923
Death Year: 1999
John Smith description"

Allow users to register.

Endpoint /users
HTTP Method POST
Query Parameters NONE
Request Body Format A JSON object containing name, email, and username.

"Please enter the following information.
Name:
Email:
Desired Username: "
Response Data Format A JSON object with the name, email, and username of new account.

"Account has been registered!
Name: John Smith
Email: johnsmith@gmail.com
Username: JohnSmith123"

Allow users to update their username.

Endpoint /users/:username
HTTP Method PUT
Query Parameters :username refers to the individual username associated with the account the user wants to update.
Request Body Format A JSON object containing the username associated with the account, the current username, and the updated username.

"Please enter the following information
Email:
Current Username:
New Username:"
Response Data Format A JSON object holding the email that was used as well as the updated username.

"Email: example@gmail.com
New Username: Example123"

Allow users to add a movie to their list of favorites.

Endpoint /users/:username/favorites/:movie
HTTP Method POST
Query Parameters :username refers to the individual account the user wants to add a movie to favorites. :movie refers to the specific movie they want to add to their favorites.
Request Body Format NONE
Response Data Format Text message that the movie has been added to their favorites.

"This movie has been added to your favorites."

Allow users to remove a movie to their list of favorites.

Endpoint /users/:username/favorites/:movie
HTTP Method DELETE
Query Parameters :username refers to the individual account which the user wants to edit their favorites. :movie refers to the specific movie they want to remove from their favorites.
Request Body Format NONE
Response Data Format Text alert that the movie has been removed from their favorites.

"This movie has been removed from your favorites."

Allow users to deregister their account.

Endpoint /users/:username/delete
HTTP Method DELETE
Query Parameters :username refers to the specific username associated with the account the user wants to delete.
Request Body Format A JSON object containing the username and email they wish to deregister

"Please enter the following information
Username:
Email:"
Response Data Format Text alert that their email has been removed.

"Your email has been removed"
