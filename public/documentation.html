Documentation for My Movie API

My Movie API is a program to made with Javascript to prove competency with Javascript, Modules, and creating/implementing API's.









Return a list of all movies to the user.

Endpoint /movies
HTTP Method GET
Query Parameters NONE
Request Body Format None
Response Data Format A JSON object holding data about all movies.
{
"Genre": {"Name": "Description":},
"Director": {"Name":, "Bio":, "Birth":, "Death":},
"_id":,
"Title":,
"Description":,
"ImagePath":,
"Featured":
}

Return data about a single movie.

Endpoint /movies/:Title
HTTP Method GET
Query Parameters :Title refers to the title of the specific movie the user is wanting information on.
Request Body Format None
Response Data Format A JSON object holding data about the specific movie.
{
"Genre": {"Name": "Description":},
"Director": {"Name":, "Bio":, "Birth":, "Death":},
"_id":,
"Title":,
"Description":,
"ImagePath":,
"Featured":
}

Return data about a movies genre.

Endpoint /movies/:Title/genre
HTTP Method GET
Query Parameters :Title refers to the title of the specific movie the user wants genre information for.
Request Body Format None
Response Data Format A JSON object that includes the genre information of a specific movie.
{
Name:
Description
}

Return data about a director of a specific movie.

Endpoint /movies/:Title/director
HTTP Method GET
Query Parameters :title refers to the title of the specific movie the user wants director information for.
Request Body Format None
Response Data Format A JSON object that includes the birth year, death year, and description of the director.
{
"Name":,
"Bio":,
"Birth":,
"Death":
}

Allow users to register.

Endpoint /users
HTTP Method POST
Query Parameters NONE
Request Body Format A JSON object containing Name, Username, Password, Email, and Birthday.
{
"Name": "",
"Username": "",
"Password": "",
"Email": "",
"Birthday": ""
}
Response Data Format A JSON object with the information of new account.
User has been added!
{
Name: '',
Username: '',
Password: '',
Email: '',
Birthday: ,
FavoriteMovies: [],
_id: ,
__v:
}

Allow users to update their username.

Endpoint /users/:Username
HTTP Method PUT
Query Parameters :username refers to the individual username associated with the account the user wants to update.
Request Body Format A JSON object containing the information that needs to be updated.
{
"Username" : "",
"Password": "",
"Email": "",
"Birthday": ""
}
Response Data Format A JSON object holding the updated user information.
Information has been updated!
{
_id: ,
Name: '',
Username: '',
Password: '',
Email: '',
Birthday: ,
FavoriteMovies:[]
}

Allow users to add a movie to their list of favorites.

Endpoint /users/:Username/movies/:MovieID
HTTP Method POST
Query Parameters :Username refers to the individual account the user wants to add a movie to favorites.
:Movie refers to the specific movie they want to add to their favorites.
Request Body Format NONE
Response Data Format JSON showing the user information with updated Favorite Movies.
{
"_id": "",
"Name": "",
"Username": "",
"Password": "",
"Email": "",
"Birthday": "",
"FavoriteMovies": []
}

Allow users to remove a movie to their list of favorites.

Endpoint /users/:Username/movies/:MovieID
HTTP Method DELETE
Query Parameters :Username refers to the individual account which the user wants to edit their favorites.
:Movie refers to the specific movie they want to remove from their favorites.
Request Body Format NONE
Response Data Format JSON showing the user information with updated Favorite Movies.
{
"_id": "",
"Name": "",
"Username": "",
"Password": "",
"Email": "",
"Birthday": "",
"FavoriteMovies": []
}

Allow users to deregister their account.

Endpoint /users/:Username/delete
HTTP Method DELETE
Query Parameters :username refers to the specific username associated with the account the user wants to delete.
Request Body Format None
Response Data Format Text alert that their account has been deleted.
"Username" has been deleted.
